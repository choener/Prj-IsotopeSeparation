## Installation

Pod5 is yet not available via conda.
A creation of a conda package from the pypi pod5 (0.1.20) package was not successfull (02.05.2023).

```
conda install mamba

mamba create -n read5 -c jannessp read5
conda activate read5
pip install pod5
```
